### Features
- Pets follow their owners
- Pets are interchangeable
- Pets can be renamed
- OP Pet Removal (to help clear lagg)

### Unimplemented Features
- Pet Riding
- Pet Inventories
- Certain Pets Don't move as they are supposed to
